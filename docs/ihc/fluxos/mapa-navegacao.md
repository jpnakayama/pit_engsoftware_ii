## Fluxo Principal
Login → Vitrine → Carrinho → Checkout (Pix) → Pedido/Status → Avaliação

## Observações
- Atalhos: Vitrine → Carrinho; Pedido (via link com `?id=` após checkout).
- Fidelidade acessível a qualquer momento (menu): mostra saldo/extrato.
- Mensagens de erro em cada etapa com linguagem simples.
